# cholesterol_analysis
A project with the goal to conduct an analysis over the cholesterol levels among a specific range of ages of men. The data is collected from official sites and the result will be presented as a graph.

# Data

The data is composed of 2 files:

- The cholesterol data: containing the id of the patients and the test results
- The demographic data: containing the patients id along with their informations (age, sex, weight ...)

# Files

This repository contains the following files:

- The data files (XPDT files)
- The analysis notebook (task.ipynb)
- The resulted csv file
- The resulted graph image
